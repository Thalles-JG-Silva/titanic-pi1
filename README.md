# Projeto de Machine Learning — Predição de Sobrevivência no Titanic (PI1)

Este projeto tem como objetivo aplicar técnicas de aprendizado de máquina supervisionado 
(Regressão Logística e Random Forest) para prever se um passageiro do Titanic sobreviveu 
ao desastre baseado em suas características.

## Dataset Utilizado

**Fonte:** Titanic Dataset (publicamente disponível)

Arquivo utilizado:
- `data/train.csv`

Principais colunas:
- `Survived` — 0 = Não sobreviveu, 1 = Sobreviveu
- `Pclass` — Classe do ticket
- `Sex` — Sexo
- `Age` — Idade
- `SibSp` — Número de irmãos/cônjuges a bordo
- `Parch` — Pais/filhos a bordo
- `Fare` — Tarifa paga
- `Embarked` — Local de embarque
